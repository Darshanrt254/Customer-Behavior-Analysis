## Overview
This project analyzes customer shopping behavior to uncover patterns in spending, subscriptions, discounts, and product performance.  
It demonstrates a **complete analytics pipeline** ‚Äî from data cleaning and SQL analysis to business-ready dashboards.

---

## Dataset
- ~3,900 customer transactions  
- 18 features including demographics, purchase details, discounts, ratings, and shipping  
- Missing values handled during preprocessing

---

## Tools & Tech
- **Python:** Pandas, NumPy, Matplotlib, Seaborn  
- **SQL:** MySQL Server  
- **BI Tool:** Power BI  
- **Environment:** Jupyter Notebook / VS Code  

---

## Workflow
1. Load data in Python  
2. Perform EDA and data cleaning  
3. Analyze business questions using SQL  
4. Build an interactive Power BI dashboard  
5. Summarize insights in a report  

---

## Dashboard
The Power BI dashboard highlights:
- Revenue by category and age group  
- Subscriber vs non-subscriber behavior  
- Shipping type and discount impact  
- Key KPIs for quick business decisions  

---

## Key Results
- Subscribers show higher long-term value  
- Loyal customers dominate the dataset  
- Some products are highly discount-dependent  
- Express shipping users spend more on average
